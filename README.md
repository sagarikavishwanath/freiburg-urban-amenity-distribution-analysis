# freiburg-urban-amenity-distribution-analysis
# Spatial Analysis of Urban Amenities (Freiburg)

This project analyzes the spatial distribution of urban amenities in Freiburg using OpenStreetMap data and R.

## Key Features
- Data extraction using OSM
- Categorization of amenities (Food, Health, Education)
- Spatial visualization (maps and density)
- Distance analysis from city center
- Identification of underserved areas

## Tools Used
- R (osmdata, sf, ggplot2)
- OpenStreetMap

## Key Insight
Amenities are concentrated in central areas, with lower access in outer regions, indicating spatial inequality.
